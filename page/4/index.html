栈与堆

首先，我们需要知道：为什么只有Objective-C对象需要进行内存管理，而其它非对象类型（如基本数据类型）不需要我们来管理呢？

因为： Objective-C 的对象在内存中是以堆的方式分配空间的

  • 堆里面的内存是动态分配的，所以也就需要程序员手动的去添加内存、回收内存
  • OC对象存放于堆里面(堆内存要程序员手动回收，也就是 release )
  • 非OC对象一般放在栈里面(栈内存会被系统自动回收)
